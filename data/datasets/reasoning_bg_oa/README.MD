# Dataset Card for Bulgarian QnA reasoning with ~2.7K entries.

### Dataset Summary

- License: apache-2.0
- Contains Parquet of a list of instructions and answers in Bulgarian language.

Each row consists of

- INSTRUCTION
- RESPONSE
- SOURCE (reasoning_bg)
- METADATA (json with language, url, id).
